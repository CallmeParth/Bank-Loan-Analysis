# Bank Loan Analysis

Welcome to **Bank Loan Analysis** Repositry ! 🚀

Financial institutions require absolute precision when evaluating lending risk and capital recovery. This project engineers a robust analytical pipeline using "bankdata.csv" (comprising 38,576 loan records) to track key performance indicators (KPIs) like total loan applications, 
funded capital, and total interest accrued.

Crucially, the project creates automated workflows to monitor loan statuses—classifying loans into Good Loans (Fully Paid & Current) vs. Bad Loans / Non-Performing Assets (Charged Off)—enabling financial risk managers to assess default rates, identify high-risk metrics, and protect organizational profitability.

---

## 🛠️ Tech Stack & Skills Demonstrated

- **Database Infrastructure**: Microsoft SQL Server.
- **Business Intelligence Tool**: Power BI Desktop.
- **Data Transformation Engine**: Power Query Editor.
- **Analytical Formulas**: Data Analysis Expression(DAX).
- **Core Competencies**: Query Optimization, Data Integration, Handling Missing Values, Automated Reporting, Dynamic User-Interface Design.
